Storage Medic Support

Storage Medic은 macOS 저장 공간을 점검하고 정리할 수 있도록 돕는 파일 관리 도구입니다.

문의 및 문제 제보

도움이 필요하거나 오류를 제보하려면 GitHub Issues를 이용해 주세요.

Open a support issue

GitHub 계정으로 작성한 공개 이슈는 GitHub의 개인정보 처리방침과 서비스 약관이 적용됩니다. 공개를 원하지 않는 개인정보나 민감한 파일 경로는 포함하지 마세요.

문의 시 포함하면 좋은 정보

  • macOS 버전
  • Storage Medic 앱 버전
  • 문제가 발생한 화면 또는 단계
  • 가능하다면 스크린샷

앱 동작 방식

Storage Medic은 사용자가 선택한 폴더를 기준으로 스캔합니다. 정리 후보는 삭제 전에 다시 확인할 수 있으며, 기본적으로 휴지통으로 이동합니다.

iCloud, Photos, Keychain, Mail, Messages, FileProvider 등 민감한 위치는 정리 후보에서 제외하도록 설계되어 있습니다.

계정 및 결제

Storage Medic은 앱 사용을 위해 별도의 계정 가입을 요구하지 않습니다. 현재 앱 내 구입이나 구독 기능도 없습니다.
