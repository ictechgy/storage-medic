Contact and issue reports

Public support requests are handled through GitHub Issues. Please do not include sensitive personal information, full file paths, or file contents in an issue.

Open a support issue

문의 시 포함하면 좋은 정보

  • macOS 버전
  • Storage Medic 앱 버전
  • 문제가 발생한 화면 또는 단계

Helpful details to include

  • Your macOS version
  • Your Storage Medic app version
  • The screen or step where the issue occurred
  • A screenshot, if available

앱 동작 방식

선택한 폴더만 스캔 정리 전 확인 기본 휴지통 이동

Storage Medic은 사용자가 선택한 폴더를 기준으로 스캔합니다. 대용량 파일, 중복 파일, 오래된 다운로드 등 정리 후보를 보여주며, 확인한 항목만 정리합니다.

iCloud, Photos, Keychain, Mail, Messages, FileProvider 등 민감한 위치는 정리 후보에서 제외하도록 설계되어 있습니다.

How the app works

User-selected folders Review before cleanup Trash-first by default

Storage Medic scans folders you choose. It shows cleanup candidates such as large files, duplicate files, and older downloads, then cleans only the items you confirm.

Sensitive locations such as iCloud, Photos, Keychain, Mail, Messages, and FileProvider are designed to be excluded from cleanup candidates.
