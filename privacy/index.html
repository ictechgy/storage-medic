Privacy Policy

Storage Medic Privacy Policy

Storage Medic does not transmit personal data, file names, file contents, or scan results to external servers through the app.

요약

Storage Medic은 사용자의 Mac 안에서 스캔 결과를 처리합니다. 앱에는 분석, 광고, 추적 SDK가 포함되어 있지 않습니다.

데이터 수집 없음 분석 SDK 없음 광고/추적 없음

Summary

Storage Medic processes scan results locally on your Mac. The app contains no analytics, advertising, or tracking SDKs.

No data collection No analytics SDKs No ads or tracking

데이터 수집

Storage Medic은 앱을 통해 개인정보를 수집, 판매, 공유하지 않습니다.

앱은 분석 도구, 광고 추적기, 제3자 추적 SDK, 원격 사용자 프로파일링 기능을 사용하지 않습니다.

Data Collection

Storage Medic does not collect, sell, or share personal data through the app.

The app does not use analytics tools, advertising trackers, third-party tracking SDKs, or remote user profiling.

파일 접근

Storage Medic은 사용자가 선택한 폴더만 스캔합니다. 스캔 결과는 사용자의 Mac 안에서 로컬로 처리됩니다.

파일 이름, 파일 내용, 스캔 결과, 정리 기록을 외부 서버로 업로드하지 않습니다.

File Access

Storage Medic scans only folders selected by the user. Scan results are processed locally on the user’s Mac.

The app does not upload file names, file contents, scan results, or cleanup history to external servers.

정리 동작

Storage Medic은 정리 후보를 먼저 보여주고, 사용자가 확인한 항목만 정리합니다.

확인된 정리 항목은 기본적으로 휴지통으로 이동합니다.

Cleanup

Storage Medic shows cleanup candidates before any cleanup action and cleans only items the user confirms.

Confirmed cleanup items are moved to Trash by default.

민감한 위치 보호

Storage Medic은 iCloud, Photos, Keychain, Mail, Messages, FileProvider 등 민감한 위치를 정리 후보에서 제외하도록 설계되어 있습니다.

Sensitive Locations

Storage Medic is designed to exclude sensitive locations such as iCloud, Photos, Keychain, Mail, Messages, and FileProvider from cleanup candidates.

지원 문의

GitHub Issues를 통해 지원을 요청하는 경우, 사용자가 직접 제공한 GitHub 사용자 이름, 메시지, 스크린샷, macOS 버전, 앱 버전 등의 정보를 받을 수 있습니다. 이 정보는 문의에 응답하고 지원을 개선하는 데만 사용됩니다.

GitHub에 게시한 정보는 GitHub의 개인정보 처리방침과 서비스 약관에 따라 처리될 수 있습니다.
